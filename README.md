# 🎮 Tic Tac Toe Game

A simple 2-player Tic Tac Toe game with score tracking and reset functionality.

---

## 📖 **About the Project**

This is a classic Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
It is a fully functional **2-player game** where players take turns and compete while tracking their scores.

The project also includes features like resetting the game board and clearing the score, making it more interactive than a basic version.
This project is part of my journey toward becoming a full stack developer.

---

## 🛠️ **Tech Stack**

* HTML
* CSS
* JavaScript
* Git & GitHub
* VS Code

---

## ✨ **Features**

* 🎯 2-player gameplay (X vs O)
* 🧮 Score tracking system
* 🔄 Reset game board option
* 🧹 Reset score option
* ⚡ Instant win detection
* 🎨 Clean and simple UI
